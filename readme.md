### Usage

To start an application you need to pass variables to `.env` file.
#### Example:

```agsl
# PostgreSQL
POSTGRES_HOST=localhost
POSTGRES_PORT=5432
POSTGRES_DB=repair_db
POSTGRES_USERNAME=root
POSTGRES_PASSWORD=root
```
